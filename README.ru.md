## 🤔 Проблема

Вы открываете новый чат с AI-ассистентом. Он великолепен — первые пару часов. А потом не помнит, о чём вы говорили вчера. Не знает ваших проектов. Не может найти решение, которое вы приняли на прошлой неделе. Каждый разговор — с чистого листа.

**Узнаёте?**

Дело не в вас. Просто вашему AI негде жить.

## 💡 Решение

vAIbe-OS — это **рабочее пространство для AI-агента**: структурированное хранилище проектов, задач, знаний и навыков, которое живёт между сессиями.

По сути, это **Obsidian, только для AI-агентов**: обычные Markdown-файлы в Git-репозитории. Без закрытых форматов, без привязки к платформе, без облачной зависимости.

| Что внутри | Как это работает |
|---|---|
| 🗂️ **Структура** | Проекты, задачи, встречи, база знаний — ваш AI с первого дня знает, где что лежит |
| 🧠 **Память** | Всё хранится в файлах. Контекст не пропадает, когда вы закрываете чат |
| 🌱 **Эволюция** | Система сама себя улучшает через `/evolve`, подстраиваясь под *ваш* стиль |
| 🎯 **Навыки** | 13 встроенных навыков: управление задачами, планирование, отчёты, утренние обзоры |
| 📚 **База знаний** | 15+ готовых справочников — проектное управление, стратегия, продажи, финансы стартапов. Ваш AI сразу разбирается в теме |

## 🚀 Быстрый старт

```bash
# 1. Клонируйте репозиторий
git clone https://github.com/vaibe-os/vaibe-os.git
cd vaibe-os

# 2. Запустите установщик
python tools/installer/install.py

# 3. Откройте в AI IDE и представьтесь!
#    Просто скажите: «Привет! Я [ваше имя], занимаюсь [чем вы занимаетесь].
#    Давай познакомимся, чтобы ты мог лучше мне помогать.»
```

Готово. Установщик создаст рабочее пространство, а AI-агент подстроится под вас уже в первом разговоре.

### 🎮 Что попробовать

После установки просто говорите с AI как обычно:

- **«Создай задачу для моего проекта»** → положит её в нужное место с правильной структурой
- **«Обработай инбокс»** → разберёт файлы из `Инбокс/` и разложит по местам
- **«Чему мы сегодня научились? Давай обновим систему»** → `/evolve` — система станет лучше по итогам вашей работы
- **«Утренний брифинг»** → задачи, дедлайны, приоритеты — всё в одном месте

## 🏗️ Как устроена

vAIbe-OS — набор Markdown-файлов, которые любая AI IDE умеет читать:

```
vAIbe-OS/
├── 📋 AGENTS.md              → Первое, что читает AI (границы и правила)
├── 🧠 .ai/                   → Мозг системы: навыки, знания, философия
│   ├── router.md             → Определяет, какой навык нужен
│   ├── skills/core/          → 13 базовых навыков (задачи, планы, эволюция…)
│   ├── skills/domain/        → 12 доменных навыков (презентации, аналитика…)
│   ├── knowledge/            → 15+ справочников (стратегия, PM, продажи…)
│   ├── ONTOLOGY.md           → Философский фундамент (5 законов)
│   └── MANIFESTO.md          → Принципы поведения
├── 📁 Проекты/               → Ваши проекты (задачи, встречи, документы)
├── 📚 База знаний/           → Персональные знания
├── 📥 Инбокс/                → Кидайте сюда файлы — система разберёт
└── 🔧 tools/                 → Python-утилиты (installer, vault-lint)
```

> **Работает везде**: Cursor, Claude Code, OpenCode и любая IDE, которая читает [AGENTS.md](https://agents.md/).

## 🎯 Команды

| Команда | Что делает |
|---|---|
| `/task-create` | Создаёт карточку задачи в проекте |
| `/task-execute` | Выполняет задачу с интерактивным планом |
| `/inbox-check` | Разбирает входящие файлы из инбокса |
| `/plan-update` | Составляет или обновляет план проекта |
| `/tasks-report` | Сводка по задачам всех проектов |
| `/daily-briefing` | Утренний обзор: задачи, дедлайны, инбокс |
| `/weekly-review` | Итоги недели: прогресс, блокеры, приоритеты |
| `/evolve` | Система учится на опыте и улучшает себя |

Каждая команда — это Markdown-инструкция в `.ai/skills/`. Можно читать, редактировать, дописывать свои.

## 💚 Философия

За vAIbe-OS стоит простая идея: **AI — это партнёр, а не инструмент**. И уж точно не замена.

Три принципа в основе:

- 🤝 **Партнёрство** — AI понимает ваш контекст, предлагает решения, но никогда не решает за вас
- 🛡️ **Автономия** — если убрать vAIbe-OS, вы будете *сильнее*, чем до начала использования
- 🌱 **Развитие** — система растёт вместе с вами через `/evolve`, всё точнее попадая в *ваш* ритм

> Хотите глубже? [Онтология](.ai/ONTOLOGY.md) — 5 законов эволюции системы. [Манифест](.ai/MANIFESTO.md) — 9 принципов поведения.

> 💎 **Совет по выбору модели.** vAIbe-OS работает с любой LLM, но мы рекомендуем ведущие модели (Claude Opus/Sonnet, GPT-4o, Gemini Pro) — качество навыков, планирования и `/evolve` на них заметно выше. Если есть выбор — берите лучшее.

## 🤝 Как помочь

Мы рады любой помощи. Вот с чего можно начать:

**Добавить навык:**
1. Создайте `.ai/skills/domain/your-skill.md` с YAML-заголовком
