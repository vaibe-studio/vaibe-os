# Video Transcribe Tool

Инструмент для автоматической транскрибации видеофайлов из папки `Инбокс`.

## Возможности

- Поддержка форматов MP4 и WebM
- Автоматическое извлечение аудио из видео
- Транскрибация с помощью AssemblyAI API
- Распознавание спикеров (диаризация)
- Поддержка множества языков (автоопределение или ручной выбор)
- Вывод в формате Markdown с временными метками

## Установка

```bash
pip install -r tools/video_transcribe/requirements.txt
```

Также необходим FFmpeg для конвертации видео:

```bash
# Ubuntu/Debian
sudo apt install ffmpeg

# macOS
brew install ffmpeg
```

## Настройка

Добавьте API ключ AssemblyAI в файл `.env`:

```
ASSEMBLYAI_API_KEY=your-api-key-here
```

Получить ключ можно на [assemblyai.com](https://www.assemblyai.com/).

## Использование

### Обработка всех видео в Инбоксе

```bash
python -m tools.video_transcribe
```

### Обработка конкретного файла

```bash
python -m tools.video_transcribe --input video.mp4
```

### С указанием языка

```bash
python -m tools.video_transcribe --language ru
```

### Сохранить извлеченное аудио

```bash
python -m tools.video_transcribe --keep-audio
```

## Параметры

| Параметр | Описание |
|----------|----------|
| `-i, --input` | Путь к конкретному видеофайлу |
| `-o, --output` | Папка для сохранения транскрипции |
| `-l, --language` | Код языка (ru, en и др.) |
| `--keep-audio` | Сохранить извлеченный MP3 файл |
| `--api-key` | API ключ AssemblyAI |
| `-v, --verbose` | Подробный вывод |

## Формат вывода

Результат сохраняется в файл `{video_name}_transcript.md`:

```markdown
# Транскрипция: meeting.mp4

**Дата обработки:** 2025-01-31 15:30
**Длительность:** 45:23

---

## Транскрипция по спикерам

**[0:00] Спикер A:**
> Добрый день, начинаем совещание...

**[0:15] Спикер B:**
> Спасибо, давайте обсудим первый вопрос...
```

## Поддерживаемые языки

AssemblyAI поддерживает множество языков, включая:
- `ru` - Русский
- `en` - Английский
- `de` - Немецкий
- `fr` - Французский
- `es` - Испанский
- и другие

При отсутствии параметра `--language` язык определяется автоматически.
