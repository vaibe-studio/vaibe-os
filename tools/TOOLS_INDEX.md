# Tools (vaibe-os)

Папка для утилит инстанса — по аналогии с `.cursor/` (правила и команды). Сюда добавляют скрипты и конвейеры, полезные для работы с репозиторием: конвертация форматов, отчёты, интеграции и т.д.

## Общие правила

- **Модульный запуск**: по возможности `python -m tools.<tool> ...` (если у инструмента есть `__main__.py`).
- **Зависимости**: в каждом подмодуле — свой `requirements.txt` при необходимости.
- **Индекс**: после добавления или изменения инструмента обновите этот файл (назначение, команда запуска).

## Структура

Каждый инструмент — подпапка `tools/<name>/` с при необходимости:

- `__init__.py`, `__main__.py` — для вызова через `python -m tools.<name>`
- `README.md` — как установить и запустить
- `requirements.txt` — точечные зависимости

## Примеры (в инстансе)

Типичные утилиты: Markdown → PDF, PDF → Markdown, статистика Cursor, создание задач в YouTrack из встреч. Конкретный набор зависит от инстанса; шаблон оставляет папку пустой для наполнения.
