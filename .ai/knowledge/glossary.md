# Glossary — Unified Terminology Dictionary

Standardized definitions for business, product, engineering, and AI terms used across vAIbe-OS projects. Each entry includes the English term, Russian translation, concise definition, and reference to the knowledge file where the concept is explained in depth.

## When to use

- Ensuring consistent terminology across project documentation
- Resolving ambiguity in task descriptions or meeting notes
- Onboarding new team members to project vocabulary
- Agent self-reference when interpreting user requests that use domain jargon

## Conventions

- Terms are listed alphabetically within each section
- **Deep-dive** column references the `.ai/knowledge/` file where the concept is explained in context
- Russian translations reflect common professional usage in the Russian tech/startup ecosystem
- Entries marked `—` for Russian have no widely established translation; the English term is used directly
- This glossary is maintained additively: new terms are added, existing terms are refined, never deleted without migration (Ontology Law 5)
