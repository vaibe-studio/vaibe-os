## Формат task.md

Каждый `task.md` ОБЯЗАТЕЛЬНО содержит секцию `## Статус` **в конце** файла:

```markdown
## Статус
- **Статус**: открыта | в процессе | выполнена | на холде
- **Завершена**: YYYY-MM-DD
- **Версия результата**: v{N} | inline | inline (описание)
```

## Результаты

- Сохранять в `results/v{N}/` — каждое выполнение в отдельной подпапке
- Предыдущие версии НЕ затирать
- Алгоритм: найти max N в `results/`, создать `v{N+1}/`

## Нумерация

- Формат: `{NUM}-{TITLE}` (NUM = ведущие нули: 001, 002, 010)
- Номера уникальны в рамках проекта
