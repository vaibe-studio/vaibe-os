# /weekly-operating-sheet

> Claude command wrapper. Canonical skill: `.ai/skills/core/weekly-operating-sheet.md`

## Quick Reference

1. Ask scope: which projects / workstreams are included
2. Ask week mode: focused / balanced / firefighting / light
3. Ask 1-3 must-win outcomes for the week
4. Gather only relevant recent meetings, active tasks, and latest plans
5. Separate: `моё решение` / `делегировано` / `жду от других`
6. Build weekly operating sheet using `База знаний/Образец плана — управленческая неделя.md`
7. Show structure and target path → confirm
8. Save to `Планы/`

## Use when

- Need a cross-project weekly management sheet
- Need owner map + waiting list + daily log
- Working in "secretary to manager" mode during the week

## Do not use when

- Need single-project sprint plan → `/plan-update`
- Need retrospective review → `/weekly-review`

## Execution

Read and follow `.ai/skills/core/weekly-operating-sheet.md` for the full procedure.

## Claude-specific enhancements

- Ask short managerial questions instead of broad discovery
- Prefer `Проекты/vAIbe-studio/Планы/` for cross-project managerial weeks
- Keep the top screen readable in 30 seconds
