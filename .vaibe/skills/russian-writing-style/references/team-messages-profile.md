# Team-message profile

> A captured profile for writing team-chat messages (lead → team).
> Captured via `/evolve` by comparing an AI draft with the user's final team-chat message.
> Apply when the task is to write an announcement/summary for the team. The message itself
> is in Russian — the examples below are kept in Russian.

## General profile

| Parameter | Value | Example |
|-----------|-------|---------|
| Formality | Medium: professional vocabulary + warm short name forms | "нижесказанным" + "через Сашу" |
| Detail | High on facts, low on rationale | Exact time (12:00), date — yes; "why this is needed" — no |
| Emotionality | Low in text, emoji = structural markers | 💡📞✏️↗️📌 — semantic, not decorative |
| Tempo | High: compact, no repetition | One thought = one bullet |

## Rules for generating team messages

1. **Sections = emoji + text on the same line**, not `###` headings. Each emoji carries meaning (📞 for calls, ✏️ for the tracker, 📌 for assignments).
2. **Numbered lists** inside sections — for referenceability ("see rule 2 in the YouTrack section").
3. **No justifications or "why" explanations** — rules are stated as fact, not as a request. The team is mature; the motivation was explained on the call.
4. **Action-first** — "Календарь обновлён" (already done), not "нужно обновить" (a request). Leadership through fact.
5. **Concrete numbers**: meeting times (12:00), dates (13.02), estimate format ("в часах/днях"). No vague "this week".
6. **Lead goals = area of responsibility**, not a list of tickets. Publicly fix the role ("обеспечение команды планированием"), not a checklist ("ревью задачи #129").
7. **Telegram handles** (@username), not full names — practical + guarantees notification delivery.
8. **Your own (PM) tasks are not published** in team messages — focus on the team.
9. **No `**bold**` in the body** — structure is set by emoji + numbering, not formatting.
10. **Closing line** — short, open: "Если что-то забыл или есть вопросы — пишите сюда."

## Anti-patterns (what NOT to do)

- Don't write "без исключений", "обязательно для всех" — an aggressive tone
- Don't explain the rules' motivation in the message itself (that's for the call)
- Don't list technical tickets (#129, #156) — that's the YouTrack level, not the chat
- Don't use the greeting "Команда," — get straight to business
