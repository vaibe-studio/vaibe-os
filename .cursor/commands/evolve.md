# /evolve

> Cursor command wrapper. Canonical skill: `.ai/skills/core/evolve.md`

## Quick Reference

1. Classify insight: type (knowledge/practice/optimization/fix), area, impact
2. Check against `.ai/MANIFESTO.md` — alignment required
3. Check for degradation — no knowledge loss, no oversimplification
4. Show plan (files to change, what changes, what stays) → **STOP** for significant changes
5. Execute changes
6. Document: what changed, why, files affected

## Trigger priorities

| Priority | When | Action |
|---|---|---|
| 🔴 Critical | Knowledge loss, contradictions, errors | Immediate |
| 🟠 High | User insights, gaps, external insights | End of current task |
| 🟡 Medium | Process improvements, patterns | Convenient moment |
| 🟢 Low | Confirmations, new understanding | Batch |

Read and follow `.ai/skills/core/evolve.md` for the full procedure.
