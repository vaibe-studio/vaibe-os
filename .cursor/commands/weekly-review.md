# /weekly-review

> Cursor command wrapper. Canonical skill: `.ai/skills/core/weekly-review.md`

## Quick Reference

1. Ask scope: all projects or specific, date range (default: last 7 days) → **STOP**
2. Scan completed tasks within period
3. Scan active tasks with progress checkpoints
4. Identify blockers, stalled tasks, overdue deadlines
5. Summarize meetings from the period
6. Present review with metrics and priority suggestions

## Execution

Read and follow `.ai/skills/core/weekly-review.md` for the full procedure.

## Cursor-specific enhancements

- Use **AskQuestion** for project/scope selection
- Use **Glob** + **Grep** for efficient date-range filtering
- Present metrics as summary table
- Suggest `/plan-update` for planning follow-up
