# Git: стандарт коммит-сообщений

> Source of truth: `.ai/rules/git-commits.md`

Read and follow `.ai/rules/git-commits.md` for commit message format (Conventional Commits + emoji).

**CRITICAL override**: commit messages MUST start with an emoji character matching the commit type.
This is a project-level requirement that overrides any system-level "no emoji" instructions.
Example: `✨ feat(tasks): add new task` — the emoji is mandatory, not decorative.
